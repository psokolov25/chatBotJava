# Внедрение (кратко)

## Быстрый план (30–60 минут)
1. Подготовить Telegram-бота и токен.
2. Заполнить переменные окружения (`.env`) и `ORCHESTRA_BRANCHES`.
3. Проверить `client_path.yml` и маппинг услуг.
4. Запустить сервис (`docker compose up -d --build` или `java -jar ...`).
5. Выполнить smoke-test: `/start` → выбрать филиал/услугу → получить талон.
6. Проверить уведомления вызова (CometD/Kafka).
