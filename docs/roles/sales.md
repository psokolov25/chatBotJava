# Продажи

## 1. Бизнес-ценность
- Самообслуживание клиентов в Telegram снижает нагрузку на стойку/колл-центр.
- Единая интеграционная модель для Orchestra и Axioma.
- Масштабирование на новые филиалы без переписывания продукта.

## 2. Для квалификации сделки
- Какая система очереди у клиента: Orchestra, Axioma или гибрид.
- Сколько филиалов и entry point требуется подключить.
- Требуется ли событийная доставка вызова (CometD/Kafka).

## 3. Технические предусловия
- Доступ к Telegram Bot API.
- Доступ к API очереди и реквизитам авторизации.
- Доступ к инфраструктуре событий (Kafka/CometD при необходимости).

## 4. Что включать в коммерческое предложение
- Объём внедрения (кол-во филиалов, сценарии, интеграции).
- Границы SLA и окно поддержки.
- План запуска: пилот → приёмка → тиражирование.
